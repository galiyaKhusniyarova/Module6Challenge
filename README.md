# Module6Challenge

The task is to create:
1. A Jupyter notebook that contains your analysis of the housing rental market data for San Francisco. 
2. The analysis will be complete with professionally styled and formatted interactive visualizations.

# Technologies
- python 3.9.12
- JupyterLab
- libraries: pandas, hvplot, pathlib, cartopy
- MacOs
# Installation
If you want to run the program yourself and/or enter different data, do the following:
1. To install the project files, go to the GitHub page for the workshop, click on the green Code button, then download the repository as a ZIP file.
